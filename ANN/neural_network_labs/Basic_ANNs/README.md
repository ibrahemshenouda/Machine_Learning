# Basic ANN Classification Labs

This folder contains notebooks implementing feedforward artificial neural networks (MLPs) using TensorFlow and Keras.

## Notebooks
* **`ANN_Sensor_Readings_4.ipynb`**: Multi-class classification using Keras Sequential models on 4-feature robot sensor readings.
* **`ANN_Robot_Movement_24.ipynb`**: Multi-class movement classification model utilizing 24-feature robot sensor readings.
* **`ANN_Breast_Cancer_Keras.ipynb`**: Binary classification network implementing Keras and standard scikit-learn preprocessing on the breast cancer dataset.
* **`ANN_ClassData_Keras.ipynb`**: Multi-class classification network implementing labels generation and model tuning on the classData faults dataset.
