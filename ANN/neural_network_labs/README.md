# Neural Network Labs

This directory contains Jupyter notebooks exploring Artificial Neural Network architectures and models.

## Structure
* [Basic_ANNs/](./Basic_ANNs): Multi-layer Perceptron (MLP) classification models using Keras and TensorFlow.
* [Autoencoders/](./Autoencoders): Autoencoder architectures for image compression, denoising, and reconstruction-based anomaly detection.
