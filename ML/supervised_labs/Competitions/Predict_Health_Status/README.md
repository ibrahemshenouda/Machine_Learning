# Predict Health Status Competition

This directory contains submissions and experiments for the Kaggle Playground Series Health Status Prediction challenge.

## Dataset
* **Source:** Kaggle Playground Series s6e2
* **Task:** Classification challenge to predict health status indicators from anonymized user metrics.

## Submissions
* **`Predict_Health_Status_Competition.ipynb`**: Modeling workflow utilizing Random Forest and LightGBM Classifiers with hyperparameter optimization.
