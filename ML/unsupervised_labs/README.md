# Unsupervised Learning Labs

This directory contains Jupyter notebooks exploring unsupervised machine learning techniques, divided into clustering and dimensionality reduction topics.

## Structure
* [Clustering/](./Clustering): Experiments with partition-based, probabilistic, and fuzzy clustering models.
* [Dimensionality_Reduction/](./Dimensionality_Reduction): Dimensionality reduction techniques to extract features and compress datasets.
