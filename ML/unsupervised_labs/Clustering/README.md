# Clustering Labs

This folder contains practical implementations of various clustering models on dataset distributions.

## Notebooks
* **`KMeans_GMM_FuzzyClustering.ipynb`**: Comparison of three clustering paradigms on the breast cancer dataset:
  * **K-Means Clustering** (hard partitioning)
  * **Gaussian Mixture Models (GMM)** (probabilistic soft clustering)
  * **Fuzzy C-Means (FCM)** (fuzzy partition-based clustering)
